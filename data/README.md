# Data Preparation

This project uses three public face analysis datasets:

- FER2013
- CelebA
- UTKFace

Due to dataset license restrictions, the original datasets are not redistributed in this repository. Please download the datasets from their official sources and organize them under this directory.

The recommended data structure is:

```text
data/
├── FER2013/
├── CelebA/
└── UTKFace/
